Marx is a classless CSS framework (perfect for Communists).

No Javacript. No Classes. Just raw CSS.

Key Features

  • Built atop of Sanitize.css.
  • Clean typography.
  • Custom forms, buttons and navigation.
  • Responsive.
  • Drop-dead gorgeous.
  • 7.7 kb minified.
For more info, check out Marx on Github.
